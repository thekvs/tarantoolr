Buffer for requests¶

In case you don’t need networking, but you need to be able to parse replies or write requests.

Create buffer¶

struct tnt_stream *tnt_buf(struct tnt_stream *s)¶

For creation of stream buffer.

struct tnt_stream *tnt_buf_as(struct tnt_stream *s, char *buf, size_t buf_len)¶

For creation of immutable stream buffer from buffer buf. It can be used for response parsing.

Create requests¶

It’s done with basic functions:

  • struct tnt_request/tnt_request_compile()
  • tnt_select()/tnt_insert() and e.t.c.

Parse response¶

Use iterator to iterate through responses or stream->read_reply(struct tnt_stream *stream, struct tnt_reply *reply).
